Channel Management Commands

Here is a list of all ChannelBot's channel management commands with usage, examples, and more! How could it be this simple?

Useful Tip:

All commands start with the ch! prefix, so a command can be run by saying ch!(command) in your discord server.


Welcome User Command


Description

Set the channel where new users will be welcomed with an embed!


Usage
ch!setwelcome (welcome-message)
ch!delwelcome

Example

ch!setwelcome Welcome to the server! - Welcomes new users to the server with the message: "Welcome to the server!".

Channel Description Command


Description

Set the description of many channels at once.


Usage
ch!description (description) | (channel mention(s))

Example

ch!description Hi! | #general #announcments - set the description/topic of general and announcments to Hi!.


Image

Channel Permission Command


Description

Quicly add a permission to a role or user in a specific channel.


Usage
ch!addperm (perm-name) (user/role-mention)

Aliases
addchannelperm + giveperm + setperm

Example

ch!addperm SEND_MESSAGES @Codingpro - Allow codingpro to send messages in the current channel.


Image

Channel Permission Remove Command


Description

Quicly remove a permission from a role or user.


Usage
ch!delperm (perm-name) (user/role-mention)

Example

ch!delperm SEND_MESSAGES @Codingpro - revent codingpro from sending messages.

In-Channel Commands

In-Channel Commands are commands that can be run in channels. These commands do not effect channels in any way, but they are still useful.

Useful Tip:

All commands start with the ch! prefix, so a command can be run by saying ch!(command) in your discord server.


Rate Command


Description

Like poll, but the bot reacts with the scale of numbers specified! Let users rate things!


Usage
ch!rate (1-5) | (question)

Examples

ch!rate 5 | this server - Rate my server from 1-5.
ch!rate 2 | 1:like cheese 2:like apples - ask users a question.


Image

Poll Command


Description

Create a poll for users to vote on.


Usage

Example

ch!poll Do you like Codingpro? - Vote if you like Codingpro.


Image

Purge Command

Moderation Commands

Moderation Commands help keep your server safe! Use these simply instead of manually doing a lot of work!

Useful Tip:

All commands start with the ch! prefix, so a command can be run by saying ch!(command) in your discord server.


Add/Delmod Command


Description

Give a user access to all commands no matter the permissions. (Will not work on commands that require ADMINISTRATOR or MANAGE_GUILD)


Usage
ch!addmod (user-mention/role-mention)
ch!delmod (user-mention/role-mention)

Examples

ch!addmod @Codingpro - add Codingpro to the mods.
ch!delmod @Codingpro - remove Codingpro from the mods.


Image

Lockdown Command


Description

Working on the server? Lock it down for users temporarily.


Usage
ch!lockdown <on/off>

Examples

ch!lockdown on - Lock the server.
ch!lockdown off - Unlock the server.


Image

Kick Command


Description

Kick A User/Bot from your server.


Usage
ch!kick (user-id) | (reason)

Example

ch!kick @codingpro | HAHHAHA - Kick Codingpro from your server.


Image

Ban Command


Description

Ban A User/Bot from your server.


Usage
ch!ban (user) | (reason)

Example

ch!ban @codingpro | being awesome - Ban codingpro from your server (Why would you?).
